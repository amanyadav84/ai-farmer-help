The Smart Crop Recommendation System is an AI-powered web application that helps farmers select the most suitable crops based on soil and environmental conditions.

Built using Streamlit and Machine Learning, this system predicts the best crops and provides additional insights and manual price mapping system for displaying crop prices

Features
- Top 3 Crop Recommendations with confidence %
- Soil Nutrient Analysis (N, P, K, pH)
- Live Weather Integration (Temperature & Humidity)
- Smart Soil Improvement Suggestions
- Crop Price Display (Manual Data)
- User-friendly Streamlit UI

Technologies Used
-Python
-Streamlit
-Scikit-learn
-NumPy
-Joblib
-Requests (API integration)

How It Works
-Enter soil nutrient values (N, P, K)
-Provide city name for live weather (optional)
-Enable/disable live weather option
-Adjust temperature, humidity, and rainfall
-Click Predict Crop
View:
Top 3 crop recommendations
Confidence levels
Best crop suggestion
